== Description ==

ChatGPT AutoPost is a powerful plugin that uses the ChatGPT API, an AI language model by OpenAI, to automatically generate and post content on your WordPress website. Save time and effort by letting the AI create engaging content for your readers.

Features:
* Automatically generate and post content using the ChatGPT API
* Customize post frequency and category
* Easy integration with the ChatGPT API key

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/chatgpt-autopost` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings -> ChatGPT AutoPost screen to configure the plugin.
4. Enter your ChatGPT API key and set the desired post frequency and category.

== Frequently Asked Questions ==

= How can I obtain a ChatGPT API key? =

You can obtain a ChatGPT API key by signing up for an API plan on the OpenAI website.

= Can I choose the category for the generated posts? =

Yes, you can configure the plugin to post in a specific category or let it choose one randomly.

= How often will the plugin generate and post content? =

You can set the desired frequency of posting in the plugin settings.
